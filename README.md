# Cardiovascular Risk Prediction System

## Overview
Machine learning decision support system predicting cardiovascular disease risk using patient health indicators.

## Technologies
Python
Scikit-learn
Pandas
NumPy
Matplotlib
Seaborn
Streamlit

## Model Performance
Accuracy: 87.5%
Cross-validation: 81.7%

## Features
Machine learning prediction model
Interactive dashboard
Data visualizations
Real-time risk prediction

## System Architecture

Dataset → Data Processing → Random Forest Model → Streamlit Dashboard → Prediction Output

## Future Improvements

Improve model accuracy using additional datasets  

Deploy the application to a cloud platform  

Add user authentication for clinical environments  

Implement model explainability using SHAP values

## How to Run

pip install -r requirements.txt  

python model_training.py  

streamlit run app.py
